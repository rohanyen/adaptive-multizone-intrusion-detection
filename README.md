# Adaptive Multi-Zone Intrusion Detection Using Hybrid Sensor Fusion on ESP32

An ESP32-based smart intrusion detection system integrating multiple sensors for adaptive threat monitoring and real-time alert generation. The system performs hybrid sensor fusion using ultrasonic, IR, sound, vibration, and environmental sensing for enhanced security monitoring.

## Features
- Multi-zone intrusion detection
- Hybrid sensor fusion
- Real-time alert generation
- WiFi-based remote monitoring
- Telegram notification support
- LED and buzzer alert system

## Hardware Used
- ESP32
- Ultrasonic Sensor
- IR Sensor
- MPU6050
- FSR Sensor
- LDR
- Sound Sensor
- LEDs and Buzzer

## Applications
- Smart security systems
- Industrial monitoring
- Smart home automation
- IoT surveillance systems
